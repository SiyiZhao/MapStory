# 从复韩遗民到汉帝国谋臣

《史记·留侯世家》中张良行迹的时空重构（入门成果稿 v0.1）

## 成果定位

本稿是一个入门级数字史学成果草稿，目标不是直接提出成熟论文结论，而是先把《史记·留侯世家》中可抽取的张良生平事件转化为一组可检查、可修订、可地图化的数据。

当前版本完成三件事：

1. 从《史记·留侯世家》拆分出 29 条张良相关主线事件。
2. 为每条事件标注结构化时间、地点、经纬度、人物、事件说明、史料备注和初步可信度。
3. 在 MapStory Web UI 中通过 `/events/map?person=张良&limit=100` 生成张良一生地理轨迹图。

这个成果适合作为博客、课程展示、数字人文项目展示或后续学术论文的材料底稿。若要发展为期刊论文，还需要补充互证史料、地名考证、前人研究综述和更严密的论证。

## 研究问题

张良的一生通常被概括为“运筹帷幄”“功成身退”，但如果把《留侯世家》的事件按时间和地点展开，他的形象会呈现出一条很清楚的政治身份转换路线：

1. 韩国旧贵族与复仇者：新郑、淮阳、东海、博浪沙、下邳。
2. 复韩运动与刘邦集团之间的中介者：薛、颍川、雒阳、宛、武关。
3. 汉楚战争中的战略参谋：霸上、鸿门、褒中、下邑、荥阳、齐地、固陵。
4. 汉初功臣政治与宫廷政治中的介入者和隐退者：留、雒阳南宫、关中、长安、曲邮、马邑。

因此，本项目的初步问题是：

《史记·留侯世家》中张良的空间移动，如何帮助我们理解他从“复韩遗民”转变为“汉帝国谋臣”的过程？

## 初步论点

《留侯世家》中的张良行迹不是一条单纯的生平旅行路线，而是一条政治身份转换的空间线索。

前期地点集中于韩国故地、反秦复仇与亡命空间，说明张良最初的行动逻辑是“为韩报仇”。中期地点沿着刘邦入关路线和楚汉战争关键节点展开，显示张良逐渐从复韩者转为刘邦集团的战略参谋。后期地点转入雒阳、关中、长安、曲邮等汉初政治空间，张良不再以战场行动为主，而是在封功、定都、太子继承和功成身退等问题上发挥影响。

换言之，地图化之后可以看到一个传统传记阅读中不够显眼的结构：张良的身份转换，是通过一连串空间场景完成的。

## 方法说明

本项目采用“事件级时空重构”的方法。

每条事件按以下字段记录：

- `time`：结构化年份。MapStory 当前以公元 0 年为公元前 1 年，因此 `-249` 对应传统纪年的前 250 年。
- `time_note`：原文纪年、汉年号、事件前后关系或不确定性说明。
- `location_note`：古地名与现代地名对应。
- `lat` / `lon`：地图展示用坐标。
- `persons`：事件涉及人物。
- `event`：现代汉语事件概述。
- `priority`：当前 MapStory 使用的粗略可信度，`史实` 或 `史实（存疑）`。
- `remark`：史料依据和判断说明。

在研究写作中，建议把 `priority` 进一步细化为以下四级：

| 等级 | 含义 | 处理方式 |
| --- | --- | --- |
| A | 原文明载地点，现代地望相对稳定 | 可作为轨迹主点 |
| B | 原文明载地点，但现代地望仍需考证 | 可绘制，但需脚注 |
| C | 原文只给方向、区域或上下文，坐标为推定锚点 | 可绘制为不确定点 |
| D | 传说性、祭祀性或后世附会色彩较强 | 仅作叙事参考 |

## 阶段划分

### 第一阶段：韩国遗民与复仇者

时间大致从韩国灭亡前后到博浪沙事后。空间上从新郑出发，经淮阳学礼、东见仓海君，最后在博浪沙刺秦失败，并亡匿下邳。

这一阶段的关键不是“张良已经是汉臣”，而是“张良仍然是韩国旧贵族”。《留侯世家》以“五世相韩”开篇，说明司马迁首先把张良放在韩国贵族谱系中理解。博浪沙刺秦不是孤立英雄故事，而是亡国复仇行动。

### 第二阶段：复韩与归刘

陈涉起兵后，张良聚少年百余人，本欲从景驹，道遇沛公后归属刘邦。此后张良随刘邦至薛，建议项梁立韩成为韩王；又与韩王将千余人西略韩地，往来颍川。

这一阶段最有研究价值：张良并非天然属于刘邦阵营。他在复韩政治与刘邦集团之间移动，说明“佐汉”身份是在秦末反秦网络中逐步形成的。

### 第三阶段：入关路线中的参谋

刘邦从雒阳南出轘辕，攻宛，入武关，至峣关、蓝田、霸上。张良在这个过程中多次提出策略：疑兵、利诱、乘懈击秦、劝刘邦还军霸上。

这一阶段的地点形成一条很强的西进路线。张良的作用不只是单次献策，而是持续参与刘邦入关路线的风险管理。

### 第四阶段：汉楚战争中的战略节点

鸿门、褒中、下邑、荥阳、齐地、固陵构成张良在楚汉战争中的关键空间。张良在鸿门危局中沟通项伯，在褒中建议烧绝栈道，在下邑提出联络英布、彭越并任韩信，在荥阳反对复立六国，在韩信请立齐王时劝刘邦因势承认，在固陵建议分地给韩信、彭越。

这一阶段显示张良的战略视野从“复韩”转为“帝国整合”。尤其是反对复立六国，与其早年复韩身份形成张力。

### 第五阶段：汉初制度化权力中的隐退与介入

刘邦定天下后，张良封留侯，参与封功臣、定都关中、保太子等汉初政治问题。空间从战场转向雒阳南宫、关中、长安、曲邮等政治中心。

这一阶段的张良似乎“隐退”，但并非完全退出。他不再以军事行动为主，而是在制度安排和宫廷政治的关键时刻介入。

## 初步观察

### 1. 下邳是张良身份转换的枢纽

下邳同时连接三个身份：亡命者、受书者、刘邦集团成员。博浪沙失败后，张良在下邳保存自身；圯桥受书使其获得后世叙事中的“帝王师”来源；陈涉起兵后道遇刘邦，则让下邳成为他从复仇者转向政治行动者的入口。

### 2. 颍川、雒阳一线暴露“复韩”与“佐汉”的过渡

张良建议立韩成，并与韩王往来颍川，这说明他在早期仍保留明确的复韩政治目标。后来他从雒阳、轘辕、宛、武关进入刘邦西进路线，才逐渐成为刘邦集团的战略参谋。

### 3. 张良的空间重心由东方转向关中

前期地点多在韩国故地、淮泗、东海、下邳。入关后，霸上、鸿门、褒中、长安逐渐成为主要场景。这个转移与张良政治身份从旧贵族复仇者到汉帝国谋臣相互对应。

### 4. 反对复立六国是张良身份转换的关键证据

荥阳“借箸筹之”一段尤其重要。早年张良为韩报仇、建议立韩王；到楚汉战争中，他却反对复立六国后裔。这不是简单的态度矛盾，而是他的政治视野从复国转向帝国统一秩序的标志。

### 5. 后期张良的“隐退”仍发生在权力中心

张良称病、辟谷、欲从赤松子游，容易被读成退出政治。但地图显示他后期仍被长安、雒阳、曲邮等政治中心牵引。所谓功成身退，并非完全离场，而是在更低可见度的位置介入制度关键点。

## 当前不足

1. 时间仍较粗。许多事件只能定位到年份或相对先后，尚未系统对照《史记》年表与《汉书》纪传。
2. 地点坐标还需要二次校勘。尤其是仓海君、曲邮、下邑、固陵、留、黄石冢相关地点。
3. 当前事件主要来自《留侯世家》，还没有充分纳入《高祖本纪》《项羽本纪》《汉书·张良传》的异文。
4. 可信度等级目前是研究稿中的人工分级，尚未写回 MapStory 数据模型。
5. 地图仍是现代经纬度投影，未叠加秦汉政区、道路、山川、关隘等历史地理图层。
